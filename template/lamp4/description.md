PHP / Apache / MariaDB
=========

## Usage
Create a "PHP / Apache / MariaDB 10.2" DEV environement.

To view the default page after launching the template, browse to http://NAS_IP:10080

Default ports :
 - **HTTP** : 10080
 - **HTTPS** : 10443
 - **MariaDB** : 13306

 Click "Next" Button to configure your environnement...
